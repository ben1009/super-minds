Grammar Focus · 语法重点

五种疑问词用法 · 点击卡片查看例句

Where
Where do you like doing sports?
→ I like doing sports in the park.
📖 例句
Where 问地点(在哪里)
Where do you like doing sth?
  • Where do you like doing homework? → I like doing homework at home.
  • Where do you like playing football? → I like playing football at school.
When
When do you like doing exercise?
→ I like doing exercise on Sunday.
📖 例句
When 问时间
When do you like doing sth?
  • When do you like playing basketball?
  • When do you like doing your test? → I like doing my test in the evening.
  • When do you like watching films? → I like watching films on Saturday.
  • When do you like driving around? → I like driving around on Saturday morning.
  • When do you like flying a kite? → I like flying a kite in the...
Which
Which sport do you like doing the most?
→ I like doing swimming the most.
📖 例句
Which 问选择(哪一个)
Which (one) do you like...?
  • Which do you like playing, football or basketball? → I like playing...
  • Which movie do you like watching the most? → I like watching XXX the...
  • Which game do you like playing the most? → I like playing XXX game the...
  • Which dish do you like eating the most? → I like eating YUXIANGROUSI the...
  • Which sport do you like playing the most? → I like playing basketball the...
Who
Who do you like doing sports with?
→ I like doing sports with my friends.
📖 例句
Who 问人(和谁一起)
Who 作宾语时，通常说 "Who do you like doing sports with?"
→ I like doing sports with my friends.
How often
How often do you like doing yoga?
→ I like doing yoga twice a week.
📖 例句
How often 问频率
→ I like doing yoga twice a week.
Pattern
Where do you like + 动词-ing?
When do you like + 动词-ing?
Which ... do you like ... the most?
Who do you like doing sports with?
How often do you like + 动词-ing?
New Dialogue · 新对话

场景:两个小朋友想踢足球,但校队满了,他们自己组队并比赛

1 🌐 翻译
A Can I join the football team?
B Sorry, the team's full.
A: 我能加入足球队吗？
B: 抱歉，队伍满员了。
2 🌐 翻译
B You can join the table tennis club.
A No, I don't like table tennis. I like football.
B: 你可以加入乒乓球俱乐部。
A: 不，我不喜欢乒乓球。我喜欢足球。
3 🌐 翻译
A Let's start a football team!
B OK!
A: 我们组建一支足球队吧！
B: 好的！
4 🌐 翻译
B We've got our team!
A Yes!
B: 我们有自己的队伍了！
A: 是的！
5 🌐 翻译
A Do you want to play a game?
B Ha ha. It's going to be easy.
A: 你想来一场比赛吗？
B: 哈哈。这将会很容易。
6 🌐 翻译
A Come on, try harder!
The other team scores first
A: 加油，再努力一点！
场景：对方先进球了
7 🌐 翻译
B Great goal!
A Well done, Flash!
A scores a goal
B: 好球！
A: 干得好，Flash！
场景：A进球了
8 🌐 翻译
B Enjoy the football team!
A No, thank you.
B: 好好享受足球队吧！
A: 不用了，谢谢。
